DevOps Portfolio Project

This project demonstrates a beginner-friendly DevOps workflow using GitHub Codespaces, Docker, and GitHub Actions. It was created to help learners understand how CI/CD pipelines, containerization, and cloud-based development environments work together.

GitHub Codespaces allows us to develop directly in the browser with no local setup. We used it to edit and run our code seamlessly. Docker was used to containerize our application, making it portable and consistent across all environments.

These commands helped us package and run our Node.js app inside a container, ensuring it behaves the same no matter where it runs.
